# Customer Segmentation Project (ML-Based)

## Project Overview
This project uses Unsupervised Machine Learning (K-Means Clustering) to segment a customer base. The goal is to identify distinct groups based on income, age, and spending behavior to drive targeted marketing strategies.

## Key Features
- **Data Scaling:** Normalized demographic data for accurate clustering.
- **K-Means Clustering:** Segmented 300+ customers into 4 distinct persona groups.
- **Behavioral Analysis:** Analyzed the relationship between annual income and spending scores.
- **Strategic Insights:** Provided marketing recommendations for each identified segment.

## Identified Segments
1. **VIPs:** High income and high spending. Target with loyalty programs.
2. **Skeptics:** High income but low spending. Target with value-based marketing.
3. **Mainstream:** Balanced income and spending. Target with standard promotions.
4. **Impulsive:** Lower income but high spending. Target with trendy, low-cost items.

## Technologies Used
- **Python (Pandas, Scikit-Learn)**
- **K-Means Algorithm**
- **Excel (for business reporting)**
